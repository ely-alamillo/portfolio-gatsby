# Personal Portfolio

## Install

Make sure you install the needed dependencies:

```sh
cd <dir-name>
yarn/npm install
```

Then you can run it by:

```sh
yarn/npm develop
```

To build the application into static files, use:

```sh
yarn/npm build
# gatsby serve to serve from static folder
```
